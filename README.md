# Saumitra Misra Portfolio

High-end portfolio site built with Next.js, TypeScript, Tailwind CSS, shadcn-style UI primitives, and Framer Motion.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-style local components
- Framer Motion
- Vercel-ready deployment

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Build

```bash
npm run build
```

## Content

Portfolio content lives in `components/data.ts`. Static assets are served from `public/Assets`.
